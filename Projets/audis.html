Cahier des charges

Pour l'implémentation du front-end, nous l'avons développé avec HTML et CSS au départ, mais le site web est encore relativement statique et peu dynamique. Par conséquent, notre tuteur a suggéré que nous finissions par utiliser Bootstrap pour créer un site Web responsive/interactive. Nous nous sommes donc formés sur Youtube et nous nous sommes finalement familiarisés avec le framwork Bootstrap.


Nos missions

Le but du projet était de développer un site web qui s'inspirait de différents éléments du site IdealAudition.com, tels que : - Présentation des différentes boutiques et formulaire de contact

Nous avons décidé de partager les tâches du projet sous plusieurs formes

- Utilisation de boostrap

- Creation d'une base de donnés avec les informations du formulaire

- Recherche d'hergement
